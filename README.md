# Dashboard - VeriTest.LAB

Repositorio para el dashboard de seguimiento del proyecto, KPIs operativos y chatbot RAG.

### Estructura

```
dashboard/
├── 01-tecnica/       # Spec técnica (React + Next.js)
├── 02-kpis/         # Definición de KPIs
├── 03-chatbot/      # Diseño Chatbot RAG
└── 04-seguimiento/ # Seguimiento proyecto
```

### Stack

- Frontend: React + Next.js
- Backend: Next.js API Routes
- Database: Supabase
- Chatbot: RAG con OpenAI
- Hosting: Vercel
